Projects

The following are
1. Official projects supervied by facult members of UofT
2. Personal projects
3. Solutions to or extensions of problems which I have found particularly interesting



Detection of Tau Neutrinos
An ongoing project focued on optimizing machine learning algorithms to determine the sensitivity of the DUNE detector to detect tau neutrinos.

Scalar Equations
This project entails the numerical evaluation of high dimensional scalar ordinary differential equations.

Euler Angles
This problem is taken from Landau and Lifshitz' Mechanics. It involes using Euler angles to model the mechanics of a rotating elipsoid.

The Simple Pendulum
This is my submission for the final project of PHY324. I construct, analyze, and model the behaviour of a simple pendulum, both within and in the complement of the region on which the small-angle approximation remains valid.

Adaptive Numerical Integration
This project entails the numerical evaluation of highly oscillatory, complex valued integrals over two-dimensional domains using an adaptive scheme implemented in Fortran 90.

Vector Field on a Torus
This is the solution to a problem that I found particularly interesting from the textbook used in MAT267

Path Optimization
Code for computing the optimal path between an arbitrary set of points placed in the real plane using a process call adiabatic cooling.
